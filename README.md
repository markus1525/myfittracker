# myfittracker

A mobile-first, single-page personal fitness tracker that runs entirely in the browser with local storage persistence.

## Overview
- Single-file HTML app with built-in styling and logic
- Tabs for Today, Supplements, Weekly Plan, Progress, and Profile
- Stores data locally in the browser using localStorage (key: fitV3)
- Includes a lightweight PWA manifest for add-to-home-screen support

## Features
- Daily workout plan with warm-up, main workout, and cool-down sections
- Exercise details modal with images, steps, mistakes to avoid, and YouTube tutorial links
- Rest timer with presets and auto-reset
- Workout completion tracking, streaks, and history
- Water intake tracker with 12-cup goal and progress bar
- Supplement checklist with editable names and times
- Weekly plan view with editable workouts per day
- Profile with BMI calculation and ideal weight range

## Getting Started
- Open myfitnesstracker.html in any modern browser
- On mobile, add to home screen to use it like an app
- Your data stays on the device and does not sync across browsers

## Customization
- Use Edit Day in the Weekly Plan tab to adjust workouts
- Use Edit in the Supplements tab to change supplement names and times
- Use Reset All Progress in the Progress tab to clear saved data

## Notes
- Exercise images and video tutorials require an internet connection
